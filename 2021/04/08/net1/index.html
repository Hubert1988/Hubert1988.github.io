Net-路由协议

路由协议

OSPF：

  • 开放最短路径优先，ip传输，使用的协议号为89

  • 工作流程

    • 发现邻居
    • 建立邻接关系
    • 交换数据库描述分组
    • 计算路由

  • ospfV2和ospfV3的区别

    • v2是基于ipv4
    • v3是基于ipv3
    • 二者的安全机制，以及lsa基于ip的版本不同，所以具有一些差距

  • 特点：

    1. 基于链路状态
    2. 规模不受限制
    3. 避免成环
      • 区域内采用stp算法，生成无环路的图
      • 区域间采用水平分割，非骨干区域只能连接到骨干区域

  • 5大分组

    1. hello分组
      • 用于邻居发现，发送周期为10s，40s没回应即为不可达
      • 目的ip为224.0.0.0.5
    2. 数据库描述分组
    3. 链路状态请求分组
    4. 链路状态宣告分组（LSU），进行洪范发送
    5. 链路状态确认

  • 区域内的DR，BDR

    • 通过内部选举出DR，BDR，让非dr/bdr的路由器和其进行建立邻接关系，从而减小了网络的开销
    • 选举规则：首先根据优先级大的为dr（路由器默认为1，为0 的不能作为DR/BDR）相同优先级的情况下比较 router-id

  • 区域的划分：

    • 骨干区域
    • 标准区域
    • 存根区域（末梢区域）
    • 完全存根区域
    • 非存根区域

  • ospf需要的三张表

    1. 路由表
    2. 邻接表
    3. 数据库信息表

  • LSA的分类 总共有11种，但是因为报文编码的问题，有的交换机不支持，所以常用的有6种

    • 总共有六大类

        1. 区域内所有路由器都可以产生 描述路由器的链路状态
        2. 区域内的DR产生 用于宣告区域内的路由
        3. 区域捏的ABR产生 产生的是路由信息，不是链路状态信息，描述区域间的路由
        4. ABR产生 告诉外部路由位置
        5. ABR产生 宣告链路外的路由信息
        6. ASBR产生 描述区域外的路由

BGP：

  • BGP是建立在TCP之上的  端口号为179

  • 特点：

    （1）BGP是一种外部网关协议，其着眼点不在于发现和计算路由，而在于控制路由的传播和选择最佳路由；

    （2）BGP使用TCP作为其传输层协议（端口号179）,提高了协议的可靠性；

    （3）BGP是一种距离矢量路由协议，在设计上就避免了环路的发生；

    （4）BGP提供了丰富的路由策略，能够实现路由的灵活过滤和选择；

    （5）BGP采用触发式增量更新，而不是周期性的更新；

  • 基本概念：

    • BGP发言人
    • BGP对等体
      • EBGP（不同的AS）
      • IBGP（相同同的AS）
      • 关于建立邻居关系：建立IBGP邻居关系是时，一般使用loopback接口的IP地址，因为loopback接口开启后一直处于UP状态，只要保证路由可达，邻居关系就会处于稳定状态。而建立EBGP邻居关系时，一般使用直接接口的IP地址，因为EBGP是跨AS建立邻居关系，邻居关系建立之前非直连接口之间的路由不可达

  • 对等体建立过程

    • TCP建立连接
    • 发送open报文
      • open报文内容：
        • 自身AS
        • 版本：通常为4
        • hold time
        • router id
    • keepalive报文（用于保持连接）
    • 路由通告

  • 对等体建立的地址为回环地址，因为属于逻辑接口，稳定

  • BGP报文：

    • BGP存在5种不同类型的报文，分别是：
      • open：用于协商参数
      • update： 传递路由信息，可以用来发布和撤销路由
      • notification：当BGP检测到错误状态时，就会向对等体发送Notification报文，告知对端错误，之后BGP连接会立即中断。
      • keepalive：双方相互发送keepalive报文，收到对方的keepalive报文后对等体建立成功，同时后续定期发送keepalive报文用于保持连接。
      • route-refresh。

  • BGP报文

    • 报文头
      1. mark 全是1 用来标记边界
      2. lenth
      3. type 用来指明报文类型
    • 报文体
